The first project is calculator

This is the code to run the calculator:

num1 = int(input("enter first number:"))
num2 = int(input("enter second number:"))

operation = print("add/a/A/+, (S/s/-)subtract, (M/m/*)ultiply, (D/d//)divide, or (off) to exit")

user = input("enter operation: ")

while user.lower() != "off":
  if user.lower() in ["a", "+"]:
    print(num1 + num2)
  elif user.lower() in ["s", "-"]:
    print(num1 - num2)
  elif user.lower() in ["m", "*"]:
    print(num1 * num2)
  elif user.lower() in ["d", "/"]:
    if num2 == 0:
      print("Error: Division by zero")
    else:
      print(num1 / num2)
  else:
    print("invalid operation")
  user = input("enter operation: ")
